# Trabalho-Semestral-Etec-Tripz
O Tripz é um projeto web desenvolvido durante minha formação em Desenvolvimento de Sistemas na ETEC, com o objetivo de criar uma plataforma de viagens funcional e visualmente organizada.

O sistema conta com múltiplas páginas, como cadastro de usuários, contato, galeria de imagens e apresentação de serviços, proporcionando uma experiência completa de navegação. O projeto foi desenvolvido utilizando HTML, CSS, PHP e JavaScript, com foco na estruturação de layouts, interatividade e organização do código.

Este projeto contribuiu para o desenvolvimento de habilidades em front-end, além de reforçar conceitos de usabilidade e construção de interfaces web.
